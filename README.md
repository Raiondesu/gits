# Usage
<!-- usage -->
```sh-session
$ npm install -g gits
$ gits COMMAND

$ gits (-v|--version|version)
gits/0.0.0 win32-x64 node-v8.12.0
$ gits --help [COMMAND]
USAGE
  $ gits COMMAND
...
```

## `gits hello [FILE]`

describe the command here

```
USAGE
  $ gits hello [FILE]

OPTIONS
  -f, --force
  -h, --help       show CLI help
  -n, --name=name  name to print

EXAMPLE
  $ gits hello
  hello world from ./src/hello.ts!
```
